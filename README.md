# TravelGo

**TravelGo** is a web-based travel management platform designed to simplify trip planning and booking. It allows users to explore destinations, view travel packages, compare options, and book trips conveniently. The system also manages user data, bookings, feedback, and admin operations efficiently.

## Features

### User Features
- Explore popular destinations in South India.
- View detailed travel packages with prices and schedules.
- Book trips and manage itineraries.
- Provide feedback on trips and services.

### Admin Features
- Manage user accounts and bookings.
- View and track all trips and inquiries.
- Generate reports and analytics on bookings and user activity.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Java (Servlets)
- **Database:** MySQL
- **Tools:** Eclipse IDE, TomTom API (for map navigation)
